Online in-browser file encryption made simple, safe and secure.

Files are secured using AES-256 GCM with keys derived from the password using 100,000 iterations of the PBKDF2 key-derivation function. Encrypted files are stored as JSON in order to aid in the encoding and decoding of the payloads.
