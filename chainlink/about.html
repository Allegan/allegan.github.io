ChainLink

ChainLink is a fully in-browser cryptography playground. All operations are done within your browser and never on an external server. Nothing that you enter into this site is ever sent over the internet ensuring that you have full custody of your information.

This applicaiton aims to provide people with easy to use, accessible, and secure cryptography that can be used regardless of platform, OS, or browser. However, because this site relies on the SubtleCrypto API a modern browser with support is required. If you do not have access to a modern, up-to-date, browser then you have larger concerns to deal with before you worry about toy cryptography.

ChainLink aims to provide a full pipeline of operations that can be used to secure communications between two parties using only this site and an unsecure channel of communication. In order to do this ChainLink provides an implementation of the Eliptical-curve Diffieâ€“Hellman key exchange (wiki) which allows two parties to create a shared secret. That secret is then hashed with the SHA-256 hashing algorithm (wiki) in order to create a 256-bit AES-GCM key that will be used to encrypt the key generated to encrypt messages with. Messages are secured with the AES-256 (wiki) block cipher in the GCM mode (wiki) which provides both strong encryption as well as message authentication.

Instructions:

  1. Find a person that you want to communicate with.

  2. Send them a link to this website.

  3. Have them generate a secret and a public key; do the same.

  4. Have them send you their public key; send them your public key.

  5. Have them use your public key to generate a shared secret; do the same. DO NOT SHARE THIS WITH ANYONE (a shared secret is one you two share but is not to be shared).

  6. Using your shared secret generate an encryption key. Share this generated key with your partner.

  7. Using your shared key and the encryption key either encrypt or decrypt messages. Share the cipher-text only from here on out.
